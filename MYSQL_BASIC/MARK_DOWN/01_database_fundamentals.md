# Database Fundamentals

This document covers basic database concepts, DBMS, RDBMS, and relational model.

---

# 1. What is Data?

Data is raw facts and figures.

Example:
- Name
- Age
- Salary
- Marks

---

# 2. What is Database?

A database is a collection of related data stored in an organized manner.

It allows:
- Easy storage
- Retrieval
- Modification
- Deletion

---

# 3. What is DBMS?

DBMS (Database Management System) is software used to manage databases.

Functions:
- Store data
- Retrieve data
- Update data
- Delete data
- Provide security

Examples:
- MySQL
- Oracle
- SQL Server

---

# 4. What is RDBMS?

RDBMS (Relational Database Management System) stores data in the form of tables.

Features:
- Data stored in rows and columns
- Tables are related using keys
- Follows relational model

Examples:
- MySQL
- PostgreSQL
- Oracle

---

# 5. Table Structure

A table contains:

- Rows (Records)
- Columns (Fields)

Example:

| ID | Name | Age |
|----|------|-----|
| 1  | Ram  | 20  |

---

# 6. Primary Key

Primary Key:
- Uniquely identifies each record
- Cannot be NULL
- No duplicate values allowed

Example:
ID column

---

# 7. Foreign Key

Foreign Key:
- Creates relationship between two tables
- Refers to primary key of another table

---

# 8. Types of Keys

- Primary Key
- Foreign Key
- Candidate Key
- Super Key
- Composite Key

---

# 9. SQL

SQL (Structured Query Language) is used to communicate with relational databases.

Used to:
- Create tables
- Insert data
- Update data
- Delete data
- Query data

---

# Summary

- Database stores organized data.
- DBMS manages databases.
- RDBMS stores data in tables.
- Keys maintain uniqueness and relationships.
- SQL is used to operate databases.
